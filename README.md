## Cloud Music For HarmonyOS NEXT

**CloudMusic (云音汇) 是鸿蒙5.0上的网易云音乐三方客户端**

该项目基于API 13，支持 HarmonyOS NEXT 5.0

AppGallery 邀请尝鲜：https://docs.chenlvin.cc/cloudmusic/#/

### 当前支持功能：

- 云音乐账号登录
- 个人歌单获取
- 音乐云盘
- 每日推荐
- 歌曲搜索
- 歌曲播放
- 歌曲下载
