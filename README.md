## Cloud Music For HarmonyOS NEXT

**Cloud Music 是鸿蒙5.0上的音乐下载App**

用户可以通过歌单、搜索，获取歌曲的下载链接

该项目基于API 12，适用于 HarmonyOS NEXT（5.0.0.102以上）；

API使用 [NeteaseCloudMusicAPI](https://gitlab.com/Binaryify/neteasecloudmusicapi)



该项目当前为测试版本，更多功能敬请期待。



### 当前支持功能：

- 网易云账号登录
- 个人歌单获取
- 歌曲搜索
- 获取歌曲下载链接
