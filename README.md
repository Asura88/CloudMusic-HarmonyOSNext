## Cloud Music For HarmonyOS NEXT

该项目基于 OHOS API14，支持 HarmonyOS 5.0.0.123 及以上系统版本。

AppGallery邀请尝鲜：https://docs.chenlvin.cc/cloudmusic/#/

**[!] 由于AppGallery的复审机制可能会导致尝鲜应用下架，建议各位自行编译或使用hap安装。**



### 安装方式

- 通过应用市场邀请尝鲜下载（不稳定）
- 自行编译安装
- 通过本仓库release中的hap包安装



### 当前支持功能：

- 云音乐账号登录
- 个人歌单获取
- 音乐云盘
- 每日推荐
- 排行榜
- 歌曲、歌单搜索
- 歌单收藏/取消收藏
- 歌单内歌曲添加/删除
- 缓存歌曲至应用沙箱
- 播放相关功能
